Changelog for 1.2.x:
  • Feature: Thumbnails are back in the profiles modal windows. I might add an option if necessary
  • Feature: The changelog displays only if the extension got a 1.x.y type update (i.e, if you update from 1.1.x to 1.2.x you will get the changelog to pop-out, otherwise, you won't)
  • Feature: Vimeo, Dailymotion and full Instagram thumbnails support
  • Feature: Added support for lightboxes with 500px, CloudApp, Droplr, Dailymotion, Flickr, Imgur, Instagram and Vimeo
  • Bugfix:The extension should now fetch correctly Imgur galleries and albums
  • Bugfix: The extension should now output less errors in the console. At least no more "useless" errors. I'm not responsible for the TweetDeck's ones of course.
  • Bugfix: Dailymotion embeds shouldn't cause HTTPS problems anymore
  • Bugfix: Tweets shouldn't have more than one thumbnail at once anymore
  • Bugfix: Instagram videos in lightboxes should fit in the modal window now
  • Bugfix: Tumblr previews should be more accurate
